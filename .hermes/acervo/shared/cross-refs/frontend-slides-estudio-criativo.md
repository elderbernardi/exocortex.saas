# Frontend Slides e Estúdio Criativo

Frontend Slides e Estúdio Criativo são capacidades pares.

## Divisão de responsabilidade

Estúdio Criativo responde por:

- direção de arte;
- tese visual;
- moodboards;
- linguagem estética;
- exploração expressiva;
- repertório visual por contexto.

Frontend Slides responde por:

- transformar conteúdo em apresentação;
- preservar fonte Markdown;
- gerar previews A/B/C;
- produzir HTML navegável;
- exportar PDF e ZIP;
- publicar export privado no Google Drive via Personal Artifact Workspace.

## Protocolo de chamada

Frontend Slides consulta Estúdio Criativo quando:

- o usuário pede algo visualmente ousado;
- a apresentação precisa de linguagem autoral;
- o microverso ativo é `estudio-criativo`;
- a primeira leitura do conteúdo não produz uma tese visual clara;
- a tarefa envolve identidade, campanha, posterização ou estética experimental.

Estúdio Criativo chama Frontend Slides quando:

- a exploração precisa virar apresentação;
- o output final deve ser HTML/PDF/ZIP;
- a peça visual precisa de navegação, palco 16:9 ou distribuição via Drive;
- o usuário pede deck, palestra, pitch ou apresentação.

## Regra

Estúdio Criativo amplia a imaginação. Frontend Slides disciplina a entrega. O Design System mantém continuidade entre ambos.
