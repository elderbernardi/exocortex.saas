# Decisão — Ontologia Multifocal v2

Diretórios funcionais representam como o Hermes opera. Natures permanecem como vocabulário semântico para humanos e agentes, declaradas no frontmatter.

Arquivos flat de Nature estão descontinuados. O adapter LLM Wiki e novas escritas usam apenas a estrutura funcional.
