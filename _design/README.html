📖 O que é isso?

O Exocórtex Design System é o sistema visual padronizado para todos os artefatos gerados pela sua extensão cognitiva — relatórios, dashboards, apresentações, sites e documentos.

Ele funciona em cascata: existe um conjunto base de tokens (o global) que vale para todos os projetos. Quando um projeto específico tem identidade própria (como a paleta da sua marca), ele declara apenas as diferenças e herda todo o resto.

🧱 Como funciona: O global define typography, spacing e componentes. Seu projeto sobrescreve apenas as cores. O resto vem de fábrica.

✅ Regras Práticas

✅ Do

  • Usar --primary em headers, navegação e botões principais
  • Usar --accent (#e88d25 no Sales AI) para CTAs e ações
  • Usar --secondary para backgrounds de card e hover
  • Manter contraste de 4.5:1 em texto sobre fundo colorido

❌ Don't

  • Usar accent em backgrounds grandes (funciona como destaque pontual)
  • Adicionar cores fora da paleta sem justificativa documentada
  • Usar mais de 2 famílias tipográficas no mesmo artefato
  • Substituir cores semânticas (danger, success) sem razão
