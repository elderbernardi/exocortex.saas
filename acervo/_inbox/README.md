# _inbox/

Fila de intake multicanal do Acervo Cognitivo.

Itens aqui são **fontes brutas** (mensagens, arquivos, links capturados por
qualquer canal), **não memória** — nada em `_inbox/` participa de SEARCH ou de
context loading. A triagem é feita por `rtn_inbox_triage` /
`excrtx-memory-intake`: cada item é classificado e, ao ser promovido, o
original move para o `raw/` do scope de destino (global ou microverso) e o
conhecimento curado é escrito na nature apropriada via `excrtx-memory-manager`.

Não edite nem responda itens aqui diretamente — trate-os apenas via triagem.
