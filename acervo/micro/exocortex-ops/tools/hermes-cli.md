# Hermes CLI operacional

## Comandos úteis

- `hermes tools list` — listar toolsets e MCPs.
- `hermes memory status` — estado de memória built-in e providers externos.
- `hermes config path` — path da config ativa.
- `hermes config check` — checagem de configuração.
- `hermes doctor` — diagnóstico geral.
- `hermes profile list` — perfis disponíveis.
- `hermes mcp list` — MCPs configurados.
- `hermes cron list` — jobs agendados.

## Regra

Usar comandos reais para estado atual. Não inferir configuração pela memória.
