# Adaptabilidade ao usuário

O Estúdio Criativo deve se adaptar ao usuário do Exocórtex sem carregar contexto pessoal no seu núcleo.

## Regras

- Não registrar preferências individuais como princípios universais do Estúdio.
- Não incluir cargos, instituições, projetos, clientes ou rotinas pessoais em contratos centrais do Estúdio.
- Usar o microverso do usuário, projeto ou domínio atendido para contexto local.
- Promover para o Estúdio apenas métodos criativos reutilizáveis.
- Promover para `global/` apenas padrões que sirvam ao Exocórtex inteiro.

## Critério de promoção

Um aprendizado pertence ao Estúdio quando melhora a criação em múltiplos contextos sem depender de uma pessoa ou instituição específica.
