# Matriz de Aspectos do Plano - Exocórtex.IA sobre Hermes Agent
## Fonte da Verdade: Harness v0.4 README e módulos associés

Este documento mapeia cada aspecto do plano contra as fontes inventariadas.
Objetivo: identificar conflitos, lacunas, decisões pendentes e requisitos preservados.

## Legenda
- **Canônico**: O que está estabelecido no README e módulos do harness v0.4 (fonte de verdade)
- **Projeto**: Fontes do repositório exocortex.saas (PDD v2, MEMORY.md, setup.sh, etc.)
- **Hermes-Setup**: Microverso no acervo dedicado ao setup do Hermes
- **Harness-Projeto**: Microverso no acervo que contém o harness v0.4 (já coberto na canônica, mas incluído para legado)
- **Classificação**: 
  - ✅ Compatível
  - ❌ Conflito
  - ⚠️ Lacuna (fonte não aborda o aspecto)
  - 🔄 Decisão Pendente
  - 📦 Requisito Preservado (deve ser mantido)
  - 🗑️ Requisito Descartável (pode ser removido)
  - 📝 Nota / Opinião do Exocórtex
