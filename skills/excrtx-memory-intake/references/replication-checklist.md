
# Replication checklist — Personal Intake Workspace

Capacidade mínima para reproduzir o intake multicanal em outro Exocórtex-Hermes:

1. Criar `~/.hermes/acervo/_inbox/` com README operacional.
2. Instalar `~/.hermes/acervo/global/contracts/personal-intake-workspace.md`.
3. Instalar `~/.hermes/acervo/global/tools/personal-intake-workspace.md`.
4. Instalar `~/.hermes/acervo/global/tools/intake_ingest.py`.
5. Instalar a skill `exocortex/personal-intake-workspace`.
6. Atualizar `setup.sh` e bundle para incluir a capability.
7. Registrar decisão e workflow no microverso `hermes-setup`.
8. Validar ingestão mínima: texto, PDF, imagem e ZIP.
9. Validar promoção semântica para um microverso existente.
10. Preservar a separação `input -> _inbox -> acervo -> _artifacts -> publish`.
